## Adding new research

When researching a new framework, regulation, or ecosystem touch-point:

1. Create a new file: `<topic-slug>.md`
2. Open with one-line description + source URL
3. Extract definitions / concepts verbatim where possible
4. Add a "CALM Academy interpretation" section
5. Add a "Why this matters for the course" section
6. Update this README's file list
7. Link from root `CURRICULUM.md` where the research informs a chapter
