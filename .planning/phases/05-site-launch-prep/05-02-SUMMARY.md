## Sitemap vs Sidebar Behavior (Key Finding)

The plan's acceptance criterion referenced `sitemap.xml` for the order test. The Docusaurus sitemap plugin always outputs URLs in alphabetical order regardless of `sidebar_position` — this is documented Docusaurus behavior. The sidebar navigation HTML (the `menu__link` elements in each built page) is the correct artifact to verify pedagogical ordering, and it confirms correct ordering. The sitemap is alphabetically ordered as expected and does not contradict sidebar correctness.
