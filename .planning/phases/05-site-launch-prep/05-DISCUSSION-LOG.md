# Phase 5 Discussion Log

**Date:** 2026-06-16
**Phase:** 05-site-launch-prep

## Gray Areas Discussed

### Area 1: Content Wiring

**Question:** How does existing `content/` get into Docusaurus `docs/`?

**Options considered:**
- Symlink `site/docs/` → `../content/` ✓ **Selected**
- Copy files during build
- Docusaurus plugin with custom content dir

**Decision:** Symlink. Single source of truth. Symlinks tracked in git. CI runs on Linux — no Windows symlink issue.

**Illustration wiring:** Symlink `site/static/img/` → `../../illustrations/exported/`. MDX references `![alt](/img/filename.svg)` — Docusaurus resolves from `static/`.

**Sidebar:** Autogenerated from directory structure. Numeric prefixes on filenames control order.

---

### Area 2: Quiz Component

**Question:** How does `<Quiz />` work in MDX?

**Options considered:**
- Custom React component `<Quiz file="..." />` ✓ **Selected**
- Third-party quiz plugin
- Static HTML embeds

**Decision:** Custom `Quiz.tsx` component. YAML loaded via js-yaml at build time. Submit-all grading (one submit for whole quiz). On submit: score + per-question explanations revealed.

**`short_answer` handling:** Read-only textarea. Model answer revealed on submit. No freetext grading in Phase 5.

---

### Area 3: Theme & Branding

**Question:** Custom theme or default Docusaurus classic?

**Options considered:**
- Classic default, minimal customization ✓ **Selected**
- Custom theme with CALM/FINOS brand colors
- Third-party theme

**Decision:** Classic default for launch. FINOS branding deferred until post-FINOS transfer. Algolia DocSearch for search (apply after first deploy; local search plugin as dev fallback).

---

### Area 4: Deploy Strategy

**Question:** How to deploy to GitHub Pages?

**Options considered:**
- GitHub Actions native Pages (configure-pages + upload-pages-artifact + deploy-pages) ✓ **Selected**
- `gh-pages` npm package
- `peaceiris/actions-gh-pages` action

**Decision:** Native GitHub Actions Pages workflow. No extra dependencies. URL: `https://gjs-opsflo.github.io/calm-academy/`. baseUrl: `/calm-academy/`. Custom domain deferred.
