# Site

Docusaurus site rendering `content/` as the public course site.

## Status

Phase 1: scaffolding only. Phase 2: Docusaurus initialized + first chapters rendered.

## Quick start (once initialized)

```bash
cd site
npm install
npm run start
# opens http://localhost:3000
```

## Build

```bash
npm run build
# output in build/
```

## Deploy

GitHub Pages initially. Hosting decision deferred to FINOS / Linux Foundation post-MVP.

## Customization

Brand: FINOS visual identity (when transferred). Pre-transfer: neutral / generic styling.
