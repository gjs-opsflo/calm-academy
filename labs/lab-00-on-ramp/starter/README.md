# Lab 0 Starter

This directory is intentionally empty. You generate your own CALM architecture document from scratch using your AI client or CALM Studio — there is no starter file to copy.

Your goal is to create `my-system.architecture.json` in this directory by following the steps in `../LAB.md`. Describe a real system you know in 3 sentences, use calmstudio-mcp or CALM Studio to generate the CALM document, then validate it with `npx @finos/calm-cli validate -a my-system.architecture.json`.

If you get stuck at any step, refer to `../LAB.md` for detailed hints, or open `../solution/my-system.architecture.json` as a reference (the solution models a Simple Blog — your architecture will describe your own system).
