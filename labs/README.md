# Labs

Hands-on labs. Skill verification. Text teaches; labs prove.

## Structure

```
labs/
└── lab-NN-<slug>/
    ├── LAB.md              Lab instructions (Learnforge-compatible YAML + Markdown)
    ├── starter/            Starter repo
    ├── solution/           Reference solution
    ├── docker-compose.yml  (optional) ephemeral environment
    └── check.sh            Automated verification
```

Format spec: [`../docs-meta/LAB-FORMAT.md`](../docs-meta/LAB-FORMAT.md)

## Status

Phase 1: empty.
Phase 2: Lab 0 (on-ramp), Lab 1 (first architecture)
Subsequent phases: one lab per teaching chapter that needs hands-on practice.
